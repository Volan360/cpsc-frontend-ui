Getting Started

CPSC Cornerstone is a comprehensive financial management platform that helps you track your institutions, manage savings goals, and monitor transactions all in one place.

dashboard

Dashboard

The dashboard provides an at-a-glance view of your financial overview. It displays:

  • Total Balance: Combined balance across all your financial institutions
  • Active Goals: Number of savings goals you're currently working towards
  • Recent Transactions: Your latest financial activities
  • Quick Actions: Easy access to create new institutions, goals, and transactions
account_balance

Institutions

Manage all your financial institutions (banks, credit unions, investment accounts) in one place.

Creating an Institution

  • Click "Create Institution" from the dashboard or institutions page
  • Enter the institution name and opening balance
  • The current balance will be automatically calculated based on your transactions

Institution Details

  • View Transactions: See all transactions associated with this institution
  • Filter by Tags: Click on transaction tags to filter by category
  • Add Transactions: Record deposits, withdrawals, and transfers
  • Edit/Delete: Update institution details or remove institutions you no longer use
flag

Goals

Set and track savings goals to help you reach your financial objectives.

Creating a Goal

  • Click "Create Goal" and enter a name and target amount
  • Add one or more institutions to allocate funds towards this goal
  • Set allocation amounts for each institution
  • Track your progress with the visual progress bar

Goal Details

  • Progress Tracking: Visual progress bar shows how close you are to your target
  • Institution Allocations: See which institutions are contributing to this goal
  • Complete Goal: Mark a goal as complete when you've reached your target
  • Transaction History: View all transactions linked to this goal
receipt_long

Transactions

Record and categorize all your financial transactions for accurate tracking.

Transaction Types

  • Deposit: Money added to an institution (positive amount)
  • Withdrawal: Money removed from an institution (negative amount)

Transaction Features

  • Tags: Categorize transactions with custom tags (e.g., "Groceries", "Salary", "Bills"). Click tags in transaction lists to automatically filter by that category
  • Notes: Add descriptions to remember transaction details
  • Filtering: Filter transactions by date range, amount, tags, or institution
settings

Profile Settings

Manage your account information and preferences.

  • Update Screen Name: Change your display name
  • Change Password: Update your account password for security
  • Delete Account: Permanently remove your account and all associated data

Tips & Best Practices

  • Use Tags Consistently: Create a standard set of tags for better organization
  • Regular Updates: Record transactions promptly for accurate balances
  • Set Realistic Goals: Break large goals into smaller, achievable milestones
  • Review Progress: Check your dashboard regularly to stay on track
  • Filter Transactions: Use filters to analyze spending patterns by category
  • Complete Goals: Mark goals as complete to maintain motivation and see your achievements
